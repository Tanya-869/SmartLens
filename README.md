# SmartLens

SmartLens is an AI-powered image analysis system that can:
- Detect objects
- Extract text (OCR)
- Scan barcodes & QR codes

## Tech Stack
- Python
- Flask
- OpenCV
- YOLOv8
- Tesseract OCR

## Features
- Multi-task image processing
- Real-time analysis
- User-friendly interface

## Setup Instructions
1. Clone the repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py

## Note
YOLO model weights are not included. Download from:
https://github.com/ultralytics/ultralytics
